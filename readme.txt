== Description ==

Directory Listings creates a fully responsive directory of businesses, organizations, etc. It creates a simple to use admin interface to create each listing, add contact information, add a logo or other image, and create an unrestricted listing description. 

Each listing uses the address to create a link to a Google Map of the listing location. 

Major features in Business Listings include:

* Fully responsive listins
* Uses the business address to create a link to a Google Map of the business location.
* Allows for full contact information as well as an unrestricted description of each business. 


== Installation ==

Upload the Directory Listings plugin to your blog or site and Activate it. BOOM, your ready to go!
